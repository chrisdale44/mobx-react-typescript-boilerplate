# Pinterest Extension App

An app to extend the functionality of Pinterest to allow filtering and searching Pins by their hashtags.

Tech stack:

* React
* React Router
* Typescript
* Mobx
* ES6
* CSS Modules 
* Webpack
* Pinterest SDK (OAuth2)
* Pinterest API

Initial run:

* Install Node.js
* `npm install`
* `npm start`
https://localhost:3000
